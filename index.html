Pour lancer le jeu, un mini-serveur web est requis !

Il semble que vous ayez ouvert ce fichier directement. Les technologies web modernes ne fonctionnent pas de cette manière pour des raisons de sécurité.

La solution la plus simple (sans installation) :

  1. Prenez le dossier complet du jeu.
  2. Allez sur le site Netlify Drop.
  3. Faites glisser le dossier sur la page.
  4. C'est tout ! Le site vous donnera un lien fonctionnel pour jouer et partager.
